2. 極限的存在與否

不是每個極限都可以判定極限值。極限在以下三種情形時「不存在」:

(I) 極限值為正負無限大:

  • 「無限大」不是確切數值，而且也不是數學上的正式寫法 (無限大 = 不存在)。

    (II) 左右極限值不相等:

  • 也因此，如果左右極限值存在且左極限值等於右極限值，極限值就存在，倒過來也是。

    (III) 震盪:

  • 像y=sinx的函數值永遠介在-1到1之間，如果將x逼近到無限大是無法判定極限值的。

    3. 極限的四則運算

設f(x)、g(x)兩函數(函數極限值皆存在)，c為一常數。

(I) 常數可以提出來:

  • 函數先乘以常數再取極限 = 函數先取極限再乘以常數。

    (II) 四則運算可分開:

  • 函數先加/減/乘/除函數再取極限 = 函數先取極限再加/減/乘/除函數。

4. 函數的連續 (Continuous)

(I) 連續的定義:

  • 函數可不可以一筆畫畫完 (不會斷掉)。可以就叫連續函數。

(II) 連續的條件 (缺一不可):

  • 極限值存在
  • 函數值存在
  • 極限值 = 函數值

第2章 微分 (Derivatives)

1. 微分的定義與表示

一半以上的數學題目都會用「微分」來求「某一點上的切線斜率」。首先，斜率的定義叫做「兩點間的垂直距離/兩點間的水平距離」，如果兩點分別為 (x,f(x)) 與 (a,f(a))，那兩點間的斜率就是 (f(x)-f(a))/(x-a)。

如果我們將 (a,f(a)) 沿著連線貼近 (x,f(x))，貼得很近很近，近到兩個點幾乎只剩 (x,f(x)) 了…這個動作，就是對兩點斜率取極限。

現在，想像一下，如果兩點貼近到幾乎只剩 (x,f(x))，那麼兩點連線斜率不就變成了 (x,f(x)) 的切線斜率了。這個動作就叫做微分。

接著，來釐清一下微分的符號:

  • 設 y=f(x)，以下各種表示皆為「y這個函數對x作微分」。
  • 越右邊是越正式的寫法，特別是最右邊的dy/dx，數學上稱它叫「一階導數」。初學者可能比就習慣用f’(x)和y’，但這些並沒有明確表示「對哪個變數做微分」，如果函數出現多個變數時就容易混淆。所以，微分題目練熟之後，要記得改用dy/dx喔!

    2. 函數的可微 (Differentiable)

(I) 可微的意思:

  • 比方說，「y函數在這個點上可微」即為「y函數在這個點上可以微分」。

    (II) 可微的條件:

  • 左導數 = 右導數。
    (剛剛不是有提到「兩點斜率取極限」嗎? 也就是說，如果兩點斜率的「左極限值」與「右極限值」相等，這個點就可微。)

    (III) 什麼時候不可微:

  • 因為左導數要等於右導數，所以只要函數圖形出現端點、尖點，那些點就一定不可微。

    4. 微分的連鎖律 (The Chain Rule)

這個定律非常容易忘記，記得! 只要出現「函數裡包著函數要求微分」，就一定要用連鎖律:

  • 先看左邊式子的等號左邊。首先，f(x)本身是一個函數，但它外面還包了一個y=x^n次方的函數，如果設這個函數為g(x)，那這兩個函數就可以表示為g(f(x))。而如果g(f(x))對x做微分，就要先微最外面得到g’(f(x))，然後再微裡面得到f’(x)，最後全部相乘，這就是微分的連鎖律。
  • 再來，看右邊的式子。為何會多出一個dx呢? 因為f(x)裡面其實還包著y=x的函數，又因為連鎖律，所以又對這個x做微分，再相乘。只是因為dx剛好是1所以不變。
  • 延哥建議大家培養這樣的敏感度，因為dx在「積分」很常用到。
  • 總之，連鎖律的口角: 外面微一次乘裡面微一次。

    4. 積分的種類

(I) 定積分 (Definite Integrals):

  • 指定從 a 積到 b，即曲線底下從 a 到 b 的面積。
  • b 稱為「上界 (Upper Limit)」；a 稱為「下界 (Lower limit)」。

(II) 不定積分 (Indefinite Integrals):

  • 有別於定積分，它沒有上下界，其實就是寫出反導函數而已。
  • 不定積分一定要加 c (一個常數)!!。不管是要你求不定積分、還是不定積分的證明，只要題目出現不定積分，就一定要加 c。
  • 一半以上的教授都認為沒加 c 是全錯!
  • 為什麼要加 c ? : 積分出來的答案再微分回去不是會變回原函數嗎? 不是只有 F(x) 微分才是 f(x)，(F(x)+c) 微分都會是 f(x)，因為常數微分是0。所以不定積分有無限組解。

(III) 瑕積分 (Improper Integrals):

  • 以下五種都屬於瑕積分:

  • 只要上下界出現無限大，或是上下界內有值讓函數變無限大的，就要當瑕積分處理。
  • 瑕積分口角: 定積分取極限。

  • 瑕積分的題目用定積分的做法做，答案會不一樣!
